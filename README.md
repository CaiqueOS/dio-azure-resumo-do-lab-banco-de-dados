# dio-azure-resumo-do-lab-banco-de-dados

Na criação de uma máquina virtual, podemos reparar que precisamos configurar basicamente tudo, desde o sistema operacional, até questões de rede. Quando criamos um banco de dados, nós só configuramos essa instância, quem pode acessar ela, o servidor e tudo mais, nós podemos criar um novo servidor para essa instância, mas não vamos configurar ele, por exemplo. Também é interessante notar que, enquanto fazemos a criação, nos dois casos, a plataforma nos informa qual seria o valor daquele serviço.
